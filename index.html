Magic Square Puzzle

Arrange the numbers so all rows, columns, and diagonals sum to the same value

Magic Constant: 0
Range: -1000 to 1000
Hints Left: 3
Time: 00:00
Solution

How to Play

A magic square is a 3×3 grid where:

  • Each row sums to the same number (magic constant)
  • Each column sums to the same number
  • Both diagonals sum to the same number

Controls:

  • Drag and drop numbers from the pool to the grid
  • Drag numbers between cells to rearrange them
  • Click the × button to remove a number from the grid
  • Use Hint to reveal a correct number (3 per game)
  • Show Solution reveals the complete answer

Levels:

  • Simple: Small positive numbers (0-20 range)
  • Normal: Medium numbers (-50 to 50 range)
  • Hard: Larger numbers (-200 to 200 range)
  • Master: Very large numbers (-1000 to 1000 range)
